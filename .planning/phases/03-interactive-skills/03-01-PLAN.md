<task type="auto" tdd="false">
  <name>Task 3: Implement Inline Skill Detail View</name>
  <files>src/App.jsx</files>
  <action>
    **Transform the card content to show skill details when selected.**

    Inside each category card (inside the `<div className="p-6 flex flex-col">`), replace the static grid content with conditional rendering:

    ```jsx
    {/* Header - Always visible */}
    <div className="mb-6 mt-2">
      <h2 className="text-xl font-bold flex items-center gap-2">
        <span className="text-indigo-400 font-mono">$</span>
        <span className="text-indigo-300 font-mono">
          {selectedSkill?.category === category.title 
            ? selectedSkill.name 
            : category.title}
        </span>
      </h2>
      <div className="text-slate-600 text-xs font-mono mt-2">
        {selectedSkill?.category === category.title 
          ? `<-- ${selectedSkill.detail.level || 'details'}/>`
          : `&lt;!-- ${category.items.length} items --&gt;`}
      </div>
    </div>

    {/* Content - Conditional */}
    {selectedSkill?.category === category.title ? (
      // DETAIL VIEW
      <div className="flex-1 flex flex-col animate-fadeIn">
        {/* Skill icon large */}
        <div className="flex justify-center mb-6">
          <img
            src={selectedSkill.icon}
            alt={selectedSkill.name}
            className="object-contain filter brightness-110"
            style={{ width: "80px", height: "80px" }}
          />
        </div>
        
        {/* Description */}
        <div className="mb-4">
          <div className="text-slate-500 text-xs font-mono mb-1">// What is it</div>
          <p className="text-slate-300 text-sm leading-relaxed">
            {selectedSkill.detail.description}
          </p>
        </div>
        
        {/* Experience */}
        <div className="mb-4">
          <div className="text-slate-500 text-xs font-mono mb-1">// My experience</div>
          <p className="text-indigo-300 text-sm leading-relaxed">
            {selectedSkill.detail.experience}
          </p>
        </div>
        
        {/* Level badge */}
        {selectedSkill.detail.level && (
          <div className="mt-auto">
            <span className="px-3 py-1 bg-indigo-500/20 border border-indigo-500/30 rounded-full text-indigo-300 text-xs font-mono">
              [{selectedSkill.detail.level}]
            </span>
          </div>
        )}
        
        {/* Back button */}
        <button
          onClick={handleBackToGrid}
          className="mt-6 w-full py-2 bg-slate-800/50 border border-slate-700/50 rounded-lg text-slate-400 text-sm font-mono hover:bg-slate-800 hover:text-indigo-300 transition-all flex items-center justify-center gap-2"
        >
          <ChevronLeft size={16} /> ./back_to_grid.sh
        </button>
      </div>
    ) : (
      // GRID VIEW
      <div className="grid grid-cols-3 gap-3 mt-auto">
        {category.items.map((item, itemIndex) => (
          <div
            key={itemIndex}
            onClick={() => handleSkillClick(item, category.title)}
            className="group relative aspect-square flex flex-col items-center justify-center gap-2 rounded-lg bg-slate-900/50 border border-slate-700/30 p-2 hover:bg-gradient-to-br hover:from-indigo-500/10 hover:to-blue-900/10 transition-all duration-300 hover:scale-105 hover:border-indigo-500/30 cursor-pointer ui-glow"
          >
            {/* ... existing icon code ... */}
            <p className="text-[9px] font-mono text-slate-400 group-hover:text-indigo-300 transition-colors text-center leading-tight line-clamp-2">
              [{item.name}]
            </p>
          </div>
        ))}
      </div>
    )}
    ```

    - Detail view should use `animate-fadeIn` or framer-motion for smooth transition
    - Text should be readable with proper contrast (slate-300 for body, indigo-300 for highlights)
    - The large skill icon should be centered and prominent
    - Back button should be styled like terminal command
    - Keep the card height consistent (may need to adjust h-[360px] if content overflows)

    **CRITICAL:**
    - Import `ChevronLeft` from lucide-react if not already imported
    - Ensure the detail view fits within the card without scrolling
    - The transition between grid and detail should feel smooth
