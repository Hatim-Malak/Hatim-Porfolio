
    2. **Add helper functions** (place before the skills array definition):
       ```javascript
       // Handle skill icon click - show detail view
       const handleSkillClick = (item, categoryTitle) => {
         setSelectedSkill({ ...item, category: categoryTitle });
       };

       // Handle back button - return to grid view
       const handleBackToGrid = (e) => {
         e.stopPropagation(); // Prevent event bubbling
         setSelectedSkill(null);
       };
       ```

       - Find the `div` with `group relative` class inside `category.items.map`

    **CRITICAL BEHAVIOR:**
    - The `selectedSkill` state tracks which skill is currently selected (null = grid view)
    - When a skill is selected, ONLY that category's card shows the detail view
    - Other cards continue showing their grid view (no change)
    - Clicking outside the card should NOT dismiss the detail view (handled in Task 3)
    - Only the explicit back button resets the state
